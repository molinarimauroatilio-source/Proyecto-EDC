# Proyecto EDC - Sindicato
**Ecosistema Digital de Certificaciones (EDC) & Sistema de Asignación Transparente (SAT)**

## 🎯 Objetivo
Desarrollar un prototipo funcional (MVP) para la gestión de certificaciones de buceo profesional y la asignación automatizada de puestos de trabajo basada en competencias y vigencia documental.

## 🛠️ Stack Tecnológico (Validado)
- **Base de Datos:** Google Sheets (Estructura de 4 Nodos).
- **Lógica de Servidor:** Google Apps Script (GAS).
- **Interfaz de Usuario:** Web App (HTML5, CSS3 con Bootstrap 5, JavaScript).
- **Control de Versiones:** GitHub.

## 🏗️ Arquitectura de Nodos
1. **Nodo Buzo:** Carga de perfil, documentos y postulaciones.
2. **Nodo Empresa:** Carga de proyectos y requerimientos técnicos.
3. **Nodo Sindicato:** Validación de datos y gestión del SAT (Ranking).
4. **Nodo Prefectura:** Consulta de validez legal (Lectura).

## 🤖 Instrucciones para Agentes IA (Protocolo Trinity)
Cualquier código o lógica sugerida debe:
1. Respetar la simplicidad del backend en Sheets.
2. Priorizar la seguridad de datos sensibles.
3. Utilizar clases de Bootstrap 5 para el Frontend.
4. Documentar cada función para facilitar el traspaso entre modelos.
